================================================================================
                              QUICK START
================================================================================

1. Install dependencies:
   $ pip install -r requirements.txt

2. Copy environment template:
   $ cp .env.example .env

3. Edit .env with your values:
   - DATABASE_URL (PostgreSQL connection)
   - REDIS_HOST, REDIS_PORT
   - PLIVO_AUTH_ID, PLIVO_AUTH_TOKEN, PLIVO_PHONE_NUMBER

4. (Later) Initialize database:
   $ python scripts/init_db.py

5. (Later) Run verification:
   $ python scripts/verify_setup.py

6. (Later) Start the app:
   $ python app.py

================================================================================
                              KEY CONCEPTS
================================================================================

MODELS (What you learned):
- Database tables that store data permanently
- Three tables: CallLog, CallerHistory, MenuConfiguration
- Defined using SQLAlchemy ORM

REDIS (Coming next):
- In-memory database for fast temporary storage
- Stores active call sessions (while call is happening)
- Data automatically expires after TTL

SERVICES (Coming next):
- Business logic layer
- redis_service - Manage sessions
- plivo_service - Generate XML
- ivr_service - Orchestrate IVR flow

FLASK ROUTES (Coming next):
- HTTP endpoints that Plivo calls
- /webhooks/answer - Incoming call
- /webhooks/input - Digit pressed
- /webhooks/hangup - Call ended
