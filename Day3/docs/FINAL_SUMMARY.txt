═══════════════════════════════════════════════════════════════════════════════
                         DATABASE SCHEMA
═══════════════════════════════════════════════════════════════════════════════

TABLE: call_logs
├─ id (primary key)
├─ call_uuid (unique identifier from Plivo)
├─ from_number (caller's phone)
├─ to_number (your Plivo number)
├─ start_time, end_time, duration
├─ menu_path (JSON array of menus visited)
├─ user_inputs (JSON array of digits pressed)
├─ call_status (completed, abandoned, error)
└─ timestamps (created_at, updated_at)

TABLE: caller_history
├─ id (primary key)
├─ phone_number (unique identifier)
├─ total_calls, total_duration
├─ first_call_at, last_call_at
├─ preferred_language, metadata
└─ timestamps (created_at, updated_at)

TABLE: menu_configurations
├─ id (primary key)
├─ menu_id (unique identifier)
├─ parent_menu_id (for hierarchy)
├─ title, message (TTS text)
├─ digit_actions (JSON mapping 1->"sales_menu")
├─ action_type (menu, transfer, hangup)
├─ action_config (transfer details, etc.)
├─ timeout, max_digits
├─ is_active (enable/disable)
└─ timestamps (created_at, updated_at)

═══════════════════════════════════════════════════════════════════════════════
                         CALL FLOW EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

1. USER CALLS YOUR PLIVO NUMBER
   ↓
2. PLIVO DETECTS CALL
   ↓
3. PLIVO → POST /webhooks/answer
   Data: CallUUID, From, To, Direction, CallStatus
   ↓
4. FLASK RECEIVES
   ↓
5. IVR SERVICE PROCESSES
   ├─ redis_service.create_session()
   ├─ Load main_menu from database
   └─ plivo_service.generate_menu_xml()
   ↓
6. FLASK RETURNS XML
   <Response>
     <GetDigits timeout="5" action="/webhooks/input">
       <Speak>Press 1 for Sales, 2 for Support...</Speak>
     </GetDigits>
   </Response>
   ↓
7. PLIVO EXECUTES XML
   ├─ Speaks message
   ├─ Waits for digit
   ↓
8. USER PRESSES DIGIT (e.g., 1)
   ↓
9. PLIVO → POST /webhooks/input
   Data: CallUUID, Digits="1", Duration
   ↓
10. FLASK RECEIVES
    ↓
11. IVR SERVICE PROCESSES
    ├─ redis_service.get_session()
    ├─ Validate digit
    ├─ Load next menu from database
    ├─ redis_service.set_current_menu()
    └─ plivo_service.generate_menu_xml()
    ↓
12. FLASK RETURNS XML (with next menu)
    ↓
13. PLIVO EXECUTES
    (Steps 7-13 repeat for each digit)
    ↓
14. CALL ENDS (user hangs up or reaches transfer)
    ↓
15. PLIVO → POST /webhooks/hangup
    Data: CallUUID, HangupCause, Duration
    ↓
16. FLASK RECEIVES
    ↓
17. IVR SERVICE PROCESSES
    ├─ redis_service.get_session()
    ├─ Save session → CallLog table (PERMANENT)
    ├─ Update CallerHistory table (PERMANENT)
    └─ redis_service.delete_session()
    ↓
18. FLASK RETURNS 200 OK
    ↓
19. DATA SAVED IN PostgreSQL
    ↓
20. CALL COMPLETE ✓
