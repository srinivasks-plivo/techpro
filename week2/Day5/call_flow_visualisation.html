Session Lifecycle — Project 5
1
Process Prewarm
Silero VAD model loaded; CallLogger initialises SQLite (receptionist_calls.db)
2
SIP INVITE → Room Created
Plivo → SIP dispatch → agent_name="receptionist-agent" → entrypoint()
3
ReceptionistAgent(room=ctx.room) created
AgentSession with same VAD · STT · LLM · TTS stack; transcript + speech event listeners attached
4
on_enter() → call_logger.start_call()
Extracts caller identity from room participants; inserts row into calls table; generates warm greeting
5
Conversation Loop + Intent Detection
LLM routes to one of 4 function tools based on caller intent; each tool logs detected intent to DB
6
Real-time Transcript Logging
user_input_transcribed & agent_speech_committed events append JSON rows to calls.transcript
7
Call Ends → on_shutdown()
UsageCollector summary logged; call_logger.complete_call() sets end_time, duration, status="completed"
SQLite Schema — calls table (call_logger.py)
idINTEGER PRIMARY KEY AUTOINCREMENT
caller_numberTEXT DEFAULT 'unknown' — phone identity or PSTN number
room_nameTEXT NOT NULL — LiveKit room name
start_timeTEXT — ISO 8601 UTC timestamp
end_timeTEXT — set on complete_call()
duration_secondsREAL — computed on completion
transcriptTEXT — JSON array of {role, text, timestamp}
detected_intentsTEXT — JSON array of intent strings
summaryTEXT — usage summary string
statusTEXT — 'active' | 'completed'
